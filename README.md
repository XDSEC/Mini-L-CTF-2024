## 题解提交方式

点击右上角~~star~~ fork 按钮将本仓库fork到你的GitHub帐号里，然后在WriteUps文件夹中新建一个文件夹，以你的队伍名称命名，将题解放在这个文件夹里，然后对本仓库提交一个Pull Request
